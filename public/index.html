NC-News API Reference

GET POST PUT DELETE

GET Requests

/api/topics

200: Responds with an object with the key of topics and the value of an array of topic objects.

Example URL Queries: 'sort_by=article_id', 'order=asc', 'topic=cats'

(Defaults to 'sort_by=created_at' and 'order=desc')

Example Response: 

{ 
  "topics": [
    { "slug": "football",
      "description":"FOOTIE!"
    },
    { "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
]}

/api/articles/:article_id

200: Responds with an object with the key of article and the value of an article object.

Parametric Endpoints: :article_id is of type "number"

Example Response: 

{
  "article": {
    "article_id": 7,
    "title": "Using React Native: One Year Later",
    "topic": "coding",
    "author": "tickle122",
    "body": "When I interviewed for the iOS developer opening...",

    "votes": 0,
    "article_img_url": "https://images.pexels.com/photos/...",
    "comment_count": 8
  }
}

POST Requests

/api/articles/:article_id/comments

Parametric Endpoints: :article_id is of type "number"

201: Responds with the posted comment.

Example Payload and Response: 

{ 
    username: "butter_bridge", 
    body: "duck, duck, duck, duck, duck, duck... goose!" 
}

PUT Requests

/api/articles/:article_id

Parametric Endpoints: :comment_id is of type "number"

200: Responds with the updated article.

Example Payload and Successful Response: 

{ 
    inc_votes: -10 
}

DELETE Requests

/api/comments/:comment_id

Parametric Endpoints: :comment_id is of type "number"

204: Responds with status 204 and no content.
